# Analysis Config v2

`default.json` is the built-in short-video operations preset. At runtime users edit a per-video copy and may save private presets.

The schema has three scopes: `shot`, `segment`, and `overall`. Fields may be grouped with `fields`; leaf fields declare `key`, `label`, `type`, `source`, and `description`.

Supported sources are:

- `vision`: generated by `qwen3.7-plus`
- `asr`: populated only from `qwen3-asr-flash-filetrans`
- `computed`: populated by application code

ASR is the authority for transcript text and timestamps. Qwen-ASR does not expose environmental-sound or music-event recognition, so those are intentionally absent from the built-in preset.

`PROMPT_CONFIG_PATH` remains available as a deployment-time default/preset import path. Results always retain their active config snapshot.
